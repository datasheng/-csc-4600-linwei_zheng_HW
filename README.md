# NYC Taxi Trip Data EDA

## 📊 Project Overview
Exploratory Data Analysis of NYC Yellow Taxi trip data from January 2023.

## 📁 Files in this Repository
- `nyc_taxi_eda.ipynb` - Main Jupyter notebook with complete EDA
- `data/` - Folder containing data samples
- `requirements.txt` - Python packages needed

## 🔍 Analysis Performed
- 1D histograms with statistics (min, max, mean, median, outliers)
- Bar charts for categorical variables
- Side-by-side box plots
- Scatter plot matrix
- Categorical vs categorical relationships
- Categorical vs numerical relationships with color

## 📦 Requirements
- pandas
- numpy
- matplotlib
- seaborn
- pyarrow
- scikit-learn

## 📝 Instructions to Run
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the notebook
